# Fabian Alba - Portfolio

- Vue.js
- Tailwind CSS
- JavaScript
- HTML & CSS

- Responsive design
- Smooth scrolling navigation
- Modern SPA structure
- Clean and simple UI

## 📂 Project Structure

- Components-based architecture
- Reusable UI elements
- Organized sections (About, Projects, Technologies, Contact)

## 🎯 Purpose

This project was created as part of my developer portfolio to demonstrate my frontend skills and ability to build modern web applications.
